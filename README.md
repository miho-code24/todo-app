# todo-app

シンプルなToDoアプリです。ローカルストレージ（localStorage）に保存されるので、ページを更新してもタスクが残ります。

## 主な機能

- タスク追加
- タスク削除
- 完了（チェック）切り替え
- タスク編集（ダブルクリック）
- 検索（入力で絞り込み）
- 並び替え（↑↓ボタン）
- データ保存：localStorage

## 使い方

1. 入力欄にタスクを入力して追加します
2. タスクの操作
   - 完了：チェック/完了ボタン等で切り替え
   - 編集：タスクをダブルクリックして編集
   - 削除：削除ボタンで削除
   - 並び替え：↑↓ボタンで上下移動
3. 検索欄に入力するとタスクを絞り込めます（クリアボタンで解除）

## 動作環境

- 推奨ブラウザ：Google Chrome / Microsoft Edge（最新）

## ファイル構成（例）

- `index.html`
- `style.css`
- `script.js`
